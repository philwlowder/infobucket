Infobucket.com

I hope you find something here of interest. You might enjoy the gallery that includes pictures of Badin Lake, Morrow Mountain State Park, Uwharrie River, Falls Reservoir, Narrows Dam and Falls Dam.

The current version of this site is created with VS Code and Gulp.js. It is hosted on Github and deployed with Netlify. Using these tools to create a static site, I only pay for a domain name. You can find links to these tools along with other resources in the footer.

I have simplified the menu system by eliminating dropdowns. As a result, most pages no longer load any javascript. They will load faster. This change comes at the cost of some links in the navigation bar. I intend to add less important links to the sitemap in the footer. I have a contact page and would enjoy comments.

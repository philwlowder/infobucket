This page will function differently depending on the size of the users display.

Displays wider than 768 pixels will see a demonstration of the Javascript plugin TableSorter. This example uses a partial list of games from the book 500 Master Games of Chess.

Click on the headings and that column will be used to sort the table.

From the TableSorter Docs, "Sort multiple columns simultaneously by holding down the Shift key and clicking a second, third or even fourth column header!" Copyright (c) 2007 Christian Bach.

Displays below 768 pixels will lose the table sorter function. The data will be available as a list of records.

This is a quick demo of the columns & zebra widget with the "sortReset" option set to true (clicking on a sort header a third time will reset the sort)

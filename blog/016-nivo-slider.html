Keep the weight off

At the top of this page I'm using the well known Nivo Slider created by Gilbert Pellegrom .

To pause the slide show move your cursor over an image. If you are using a smart phone, I suggest landscape mode. In order for the slide show to work, you may need to go to landscape mode first and then load the page.

Images don't have to be megabyte monsters to look fine on a monitor. I'm compressing each image and deferring all but the first slide. The first image of the slide show pops into place and after the rest of the page completely loads in the browser, the remaining images load.

There are several web sites that test load times. Here I've tested Infobucket with Webpagetest .

This is not a WordPress or a Bootstrap site. I'm using the tools listed at the bottom of this page. Get in touch if you have a comment, find a mistake or just want to talk about HTML and CSS. My contact page has a mailto link.

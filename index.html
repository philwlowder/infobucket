I created this site with VS Code and Gulp. I host it on Github and deploy it with Netlify. Using these tools to create a static site, I only pay for a domain name. You can find links to these tools and a few others in the footer.

I hope you find something here to interest you. You might enjoy the gallery of pictures that include Badin Lake, Morrow Mountain State Park, Uhwarrie River, Falls Reservoir and Falls Dam. This is the place to start, please look around. I have a contact page and would enjoy comments.
